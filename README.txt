智慧科技執法安全帽偵測系統
=================================

本專案利用深度學習與 YOLO 物件偵測模型，
建立一套模擬智慧科技執法的安全帽偵測系統。

系統可透過攝影機或 RTSP 串流即時偵測機車騎士是否配戴安全帽，
若偵測到違規情況，系統會自動進行標記並截圖保存，
並將相關資訊輸出紀錄，方便後續分析與管理。

---------------------------------
專案功能
---------------------------------

1. 即時安全帽偵測
   透過 YOLO 模型辨識騎士是否配戴安全帽。

2. 違規自動標記與截圖
   偵測到未配戴安全帽時，自動標記違規並截圖保存。

3. 支援 RTSP / 攝影機串流
   可直接接收 RTSP 串流或本地攝影機畫面進行即時偵測。

4. 模型蒸餾 (Knowledge Distillation)
   透過模型蒸餾技術壓縮模型大小並提升推論速度，
   以利於即時監控應用。

---------------------------------
專案檔案結構
---------------------------------

main.py
主程式，負責執行整體偵測流程。

YOLO_mix.py
YOLO 模型偵測與相關邏輯。

config.py
系統參數設定檔。

traces.csv
偵測結果與軌跡紀錄輸出。

readme.txt
專案說明文件。

requirements.txt
Python 套件需求清單。

---------------------------------
系統需求
---------------------------------

Python 3.8 以上版本

需要安裝以下 Python 套件：

opencv-python
numpy
pandas
ultralytics
torch
torchvision

安裝方式：

pip install -r requirements.txt

---------------------------------
執行方式
---------------------------------

1. 安裝必要套件

pip install -r requirements.txt

2. 執行主程式

python main.py

---------------------------------
模型效能結果
---------------------------------

推論速度：30 FPS

mAP：97%

模型大小：由 40MB 壓縮至 5MB

透過模型蒸餾技術，大幅降低模型大小與推論延遲，
使系統更適合即時監控與邊緣運算應用。

---------------------------------
備註
---------------------------------

可透過修改 config.py 來調整系統參數，
例如影像來源、偵測設定與輸出方式。
